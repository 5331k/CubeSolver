*cubesolver is a maven based java project. It can be directly imported to eclipse workspace or can be executed via maven. 
*There is a batch file in the folder that can be used to build and then execute project via maven. 
*Once the project is successfully build and executed, Generated output is printed on console and written in file.
*All output files can be found in "output" folder. File names are generated via timestamp.
